# 记账网页应用 - 产品需求文档 (PRD)

## 一、产品概述

### 1.1 产品定位
一个简洁实用的个人记账网页应用，无广告、易上手，支持基础的收支记录、分类管理和数据统计功能。

### 1.2 核心价值
- **简洁**：无广告、无冗余功能，专注记账本身
- **实用**：覆盖日常记账核心需求，操作流畅
- **安全**：数据本地存储或云端同步，保护隐私

### 1.3 目标用户
- 个人用户，希望简单记录日常收支
- 不想使用复杂或广告过多的记账App
- 偏好网页端操作，多设备访问

---

## 二、用户分析

| 维度 | 分析内容 |
|------|----------|
| 画像 | 18-45岁，有记账需求的个人用户，具备基本互联网使用能力 |
| 场景 | 日常消费后记录支出、每月初查看上月收支情况、制定预算计划 |
| 痛点 | 现有App广告多、功能复杂、需要下载安装 |
| 期待 | 简洁界面、快速记账、数据可视化、多设备访问 |

---

## 三、竞品分析

| 竞品 | 类型 | 核心功能 | 优点 | 缺点/痛点 | 可借鉴 |
|------|------|----------|------|-----------|--------|
| 随手记 | 直接竞品 | 多账本、图表分析、预算管理、语音记账 | 功能全面、生态完善 | 广告多、功能过于复杂 | 图表分析、预算管理 |
| 挖财记账 | 直接竞品 | 智能记账、账单导入、财务报表 | 界面友好、智能分类 | 广告较多 | 智能分类 |
| 钱迹 | 直接竞品 | 记账、分类、数据同步 | 无广告、界面简洁 | 功能相对简单 | 极简设计理念 |
| 记账城市 | 间接竞品 | 游戏化记账、城市建设 | 趣味性强 | 游戏元素干扰记账核心 | 可考虑轻度激励 |
| 口袋记账 | 直接竞品 | 基础记账、简洁界面 | 轻量、启动快 | 功能较少 | 简洁设计 |

**差异化机会：**
- 无广告纯净体验
- 网页端便捷访问，无需下载
- 核心功能完整但不臃肿
- 现代化UI设计

## 四、功能设计

### 4.1 核心功能

#### 记账功能
- 快速记账：输入金额、选择分类、备注
- 收入/支出切换
- 日期选择
- 支持常用分类自定义

#### 分类管理
- 预置常用分类（餐饮、交通、购物、娱乐等）
- 支持添加/编辑/删除分类
- 分类图标和颜色标识

#### 账单列表
- 按时间倒序展示账单
- 支持按月份筛选
- 支持按分类筛选
- 支持搜索备注

#### 数据统计
- 月度收支概览
- 分类占比饼图
- 收支趋势折线图
- 月度对比分析

#### 预算管理
- 设置月度预算
- 预算进度提醒
- 超支预警

### 4.2 用户系统
- 用户注册/登录
- 密码找回
- 数据云端同步

## 六、验收标准

### 6.1 功能验收

| 功能 | 验收标准 |
|------|----------|
| 用户注册登录 | 正常注册、登录、登出，密码验证正确 |
| 记账 | 能成功添加收入/支出记录，数据正确存储 |
| 分类管理 | 能添加/编辑/删除分类，图标和颜色正确显示 |
| 账单列表 | 账单按时间倒序展示，数据准确 |
| 数据统计 | 图表数据与实际账单数据一致 |
| 预算管理 | 预算进度计算正确，超支提醒正常 |

### 6.2 性能验收

| 指标 | 标准 |
|------|------|
| 页面加载时间 | 首屏加载 < 3秒 |
| 接口响应时间 | API响应 < 500ms |
| 并发支持 | 支持100用户同时在线 |

### 6.3 兼容性验收

| 平台 | 要求 |
|------|------|
| 浏览器 | Chrome、Firefox、Safari、Edge最新版本 |
| 设备 | 支持PC和移动端响应式布局 |

### 6.4 安全验收

| 安全项 | 要求 |
|--------|------|
| 密码存储 | 加密存储，不明文保存 |
| 接口安全 | 用户数据隔离，JWT认证 |
| 数据安全 | 定期备份，支持数据恢复 |

## 七、技术方案建议

### 前端
- 框架：React + TypeScript
- UI组件：Ant Design 或 Tailwind CSS
- 状态管理：React Query + Zustand
- 图表：ECharts 或 Chart.js

### 后端
- 框架：Node.js + Express 或 NestJS
- 数据库：PostgreSQL 或 MySQL
- 认证：JWT
- API：RESTful

### 部署
- 容器化：Docker
- 前端：Nginx
- 后端：Node.js服务
