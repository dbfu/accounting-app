# 记账网页应用 - 测试用例文档

## 一、测试概述

### 1.1 测试范围
- 用户系统：注册、登录、登出、个人信息管理
- 记账功能：收支记录的增删改查
- 分类管理：分类的增删改查
- 账单列表：筛选、搜索、分页
- 数据统计：月度概览、分类占比、收支趋势
- 预算管理：预算设置、进度展示

### 1.2 测试环境
| 项目 | 说明 |
|------|------|
| 前端地址 | http://localhost:5176 |
| 后端地址 | http://localhost:3000 |
| 数据库 | PostgreSQL 16.x |
| 浏览器 | Chrome、Firefox、Safari、Edge |

### 8.3 缺陷等级定义

| 等级 | 定义 | 处理时限 |
|------|------|----------|
| 致命 | 系统崩溃、数据丢失、安全漏洞 | 立即修复 |
| 严重 | 核心功能不可用 | 24小时内 |
| 一般 | 功能异常但有替代方案 | 3天内 |
| 轻微 | UI问题、优化建议 | 下版本修复 |
